安装Hexo常见问题

发表于 2020-01-14 更新于 2020-01-15 分类于 绊脚石 Disqus：

近日由于 wp 的访问越来越慢，考虑迁移到 hexo。在安装过程中发现一些问题,记录下。

node: v12.14.1
npm: 6.13.4
os: macOS Mojava 10.14.5

Git clone 太慢，或者错误

下载速度只有 几kib/s 或者直接就报错下不下来。

网上找了一些方法，有用的有下面几个，自己根据自己的性能选择下即可。

加大缓存区

少clone一些，–depth 1

> –depth 1 复制深度为1，就是每个文件只取最近一次提交，不是整个历史版本。

从码云上 clone 【推荐】

  • 把你需要 clone 的目标在 Github 上复制地址

  • 打开码云gitee.com, 新建仓库->从 GitHub 导入

  • 在本地用码云的仓库地址 进行 clone

NPM 安装出现权限不足的问题

npm install xxx 时，因为没有权限造成的该错误。网上普片的处理方式是在命令前加 sudo,但是这样做并没有解决实际问题。一劳永逸的方法如下：

  • 修改npm包所安装目录的权限
  • 接下来可以进行npm全局包安装
