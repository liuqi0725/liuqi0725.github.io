不管启动与否都可以拷贝

命令：docker cp <image_id>:<target_dir_path> <local_dir_path>

  • image_id : 镜像 id 可以通过 docker ps -a 查看，可以简写(取 id 前几位)
  • target_dir_path : docker 中目标文件夹路径
  • local_dir_path : 拷贝到本地的路径
